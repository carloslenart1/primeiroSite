design

O design é um elemento muito importante do desenvolvimento web, para que as páginas fiquem atraentes aos nossos clientes, e representem da melhor forma os produtos e a empresa dele aos seus consumidores.

Já imaginou

Se as páginas web fossem apenas em preto e branco, sem formatação de cores, fontes, decorações de texto, imagens não dimensionadas, como poderíamos criar páginas de modo a encantar os clientes?

Formas de conexão HTML e CSS

Existem três formas para conectar um design CSS com a programação HTML.

  1. Interna
  2. Externa
  3. Por linha
Conexão externaPara o funcionamento desta forma, é necessário um arquivo externo .css, onde terá os dados que darão "vida" ao site. E a conexão ocorrerá atraves da tag < limk > e no local "href" será colocado o link ou endereço de conexão do css.
Conexão internaO interno é criado por meio da tag style, em nosso cabeçalo head, tem uma estrutura Similar ao do arquivo externo, porém reside no meu documento html.
Em linhaA conexão em linha é realizada dentro da tag de abertura que você deseja estilizar, com o atributo style="", entre as aspas passe as propriedades que você deseja utilizar, sempre separando por ponto e vírgula. EX: style="color:red; font-size:18".

Todos os comandos do css devem se finalizar com ponto e vírgula(;).

Referenciando os elementos para estilização

Os elementos podem ser referenciados de três formas diferentes:

  1. Identificação
  2. Classe
  3. Proprio nome
IdentificaçãoTodo elemento tem um id único que é uma identificação do elemento. O id é uma identificação definida na tag de abertura como id="". No css para usar o id de um elemento para estilizá-lo basta colocar #idDoElemento{}. Entre as chaves passamos os parâmetros para a estilização.
classeA classe identifica não somente um elemento, mas um grupo de elementos, desde que definamos para todos a mesma classe, pelo nome que é atribuído a ela. Assim como o id, deve ser definido na tag de abertura dos elementos: class="",geralmente definimos por seu nome a sua função . No css para usar a classe de um elemento para estilizá-lo basta colocar .classDoElemento{}. Entre as chaves passamos os parâmetros para a estilização.
Nome da tagAo utilizar o nome da tag propriamente dito, ex: p, body, h1, estilizamos todas as tags deste tipo que não possuem id, ou class. No css para usar o nome de um elemento para estilizá-lo basta colocar nomeElemento{}. Entre as chaves passamos os parâmetros para a estilização.

Importante: A ordem de precedência de aplicação de estilização por prioridade é em linha, interno e por último externo.
