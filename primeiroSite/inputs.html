Formulários e Entrada de Dados



Os formulários em html são representados pela tag input, ela possui diferentes entradas de dados: textos, números, e-mail, senhas, sexo.
Toda vez que criamos a tag input ela cria uma caixa de entrada.


A tag input tem os seguintes parametros: type, para indicar que tipos de informações será inserida (text, password, checkbox, radio, number, e-mail), bem como o campo de placeholder, é como se fosse uma dica para o usúario de qual informação precisa ser inserida ali, após clicar no campos desse texto desaparece.
